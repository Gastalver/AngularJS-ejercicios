#Ejercicios aprendizaje AngularJS
Ejercicios realizados a partir del libro AngularJS. No sólo comprenden los ejercicios, también consejos sobre IDE y Flujo de Trabajo.
#Proyecto
Elegir el directorio de trabajo. Crearlo.
#Git
Revisar la configuración de WebStorm para el proyecto en File > Settings > Version Control
Inicializar git desde Terminal.
```shell
git init
```
Alternativamente desde WebStorm: VCS se puede crear un repositorio de GitHuB
Tambien se puede usar Git, para conectar con BitBucket.

#npm
Inicializar npm desde Terminal para crear el archivo de paquetes de node.
```shell
npm init
```
Se inicia un asistente.
Webstorm: En menú contextual del archivo package.json seleccionar 'Show npm scripts' para habilitar la ventana npm.


#grunt
Instalar grunt desde Terminal
```shell
npm install grunt -S-D
```

Se puede inicializar el entorno de trabajo incluido el archivo Gruntfile.js gracias a una herramienta de la linea de comandos llamada grunt-init que hay que instalar y configurar una plantilla. Vease [http://gruntjs.com/project-scaffolding]

Cómo mínimo crear un archivo Gruntfile.js aunque sea a mano.

Webstorm: En menú contextual del archivo Gruntfile.js seleccionar 'Show grunt scripts' para habilitar la ventana Grunt con las tareas.
