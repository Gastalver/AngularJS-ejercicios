#Git
Revisar la configuración de WebStorm para el proyecto en File > Settings > Version Control
Inicializar git desde Terminal.
```shell
git init
```
Alternativamente desde WebStorm: VCS se puede crear un repositorio de GitHuB
Tambien se puede usar Git, para conectar con BitBucket.
